# DROP0 macro

This macro creates the DROP_REG macro

## Syntax

``` hlasm
         DROP0 ,
```

The DROP0 macro takes no arguments.
It is intended for use by the [DROP macro](DROP.md)

## Macro code

The [DROP0 macro](../bxamac/DROP0.mac) is in the BXAmac folder.
