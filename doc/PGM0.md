# PGM0 macro

This macro creates the [CHK_STACK macro](CHK_STACK.md)

## Syntax

``` hlasm
         PGM0 ,
```

The PGM0 macro takes no arguments.
It is intended for use by the [PGM macro](PGM.md)

## Macro code

The [PGM0 macro](../bxamac/PGM0.mac) is in the BXAmac folder.
