# BALZ macro

Branch and Link on Zero condition

## Syntax

``` hlasm
&LABEL   BALZ  &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *                                
```
