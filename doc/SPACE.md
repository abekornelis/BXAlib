# SPACE macro

This macro replaces the assembler's SPACE statement                

## Syntax

``` hlams
&LABEL   SPACE ,                       * Arguments in SYSLIST         
```

The syntax for the SPACE macro is identical to that of the SPACE instruction.

## Macro code

The [SPACE macro](../bxamac/SPACE.mac) is in the BXAmac folder.
