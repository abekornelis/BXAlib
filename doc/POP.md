# POP macro

This macro replaces the assembler's POP statement          

``` hlasm
&LABEL   POP   ,                       * Arguments in SYSLIST 
```

The syntax for the POP macro is identical to that of the POP instruction.

## Macro code

The [POP macro](../bxamac/POP.mac) is in the BXAmac folder.
