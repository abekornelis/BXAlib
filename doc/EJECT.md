# EJECT macro

This macro replaces the assembler's EJECT statement.
It implements the overrides, as specified by preceding invocations
of the [EJECTOVR macro](EJECTOVR.md).

``` hlasm
&LABEL   EJECT ,                       * Arguments in SYSLIST
```

The syntax for the EJECT macro is identical to that of the EJECT instruction.

## Macro code

The [EJECT macro](../bxamac/EJECT.mac) is in the BXAmac folder.
