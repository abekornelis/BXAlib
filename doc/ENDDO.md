# ENDDO macro

- Structured programming macro: ENDDO
- Combines with [DO macro](DO.md) to repeatedly execute a block of code.

## Syntax

``` hlasm
&LABEL   ENDDO ,                       * No arguments allowed
```

## Macro code

The [ENDDO macro](../bxamac/ENDDO.mac) is in the BXAmac folder.
