# MAPIECEQ macro

This macro maps the Open/Close/EOV equates

Corrected bit defintitions from IECEQU are used.

## Syntax

``` hlasm
         MAPIECEQ ,
```

## Macro code

The [MAPIECEQ macro](../bxamac/MAPIECEQ.mac) is in the BXAmac folder.
