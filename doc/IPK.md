# IPK macro

This macro replaces the IPK instruction

## Syntax

``` hlasm
&LABEL   IPK   ,                       * Parameters in SYSLIST
```

The syntax for the IPK macro is identical to that of the IPK instruction.

## Macro code

The [IPK macro](../bxamac/IPK.mac) is in the BXAmac folder.
