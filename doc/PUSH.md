# PUSH macro

This macro replaces the assembler's PUSH statement           

## Syntax

``` hlasm
&LABEL   PUSH  ,                       * Arguments in SYSLIST   
```

The syntax for the PUSH macro is identical to that of the PUSH instruction.

## Macro code

The [MVPL macro](../bxamac/MVPL.mac) is in the BXAmac folder.
