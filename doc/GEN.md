# GEN macro

This macro generates a replaced machine instruction

## Syntax

``` hlasm
&LABEL   GEN   &X                      * &X must be a valid instruction
.*                                     * Other operands in SYSLIST
```

## Macro code

The [GEN macro](../bxamac/GEN.mac) is in the BXAmac folder.
