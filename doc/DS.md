# DS macro

This macro replaces the assembler's DS statement
It is implements the overrides, as specified by preceding invocations
of the [DSOVR macro](DSOVR.md).

## Syntax

``` hlasm
&LABEL   DS    ,                       * Arguments in &SYSLIST
```

The syntax for the DS macro is identical to that of the DS instruction.

## Macro code

The [DS macro](../bxamac/DS.mac) is in the BXAmac folder.
