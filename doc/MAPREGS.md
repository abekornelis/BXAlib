# MAPREGS macro

This macro maps the registers onto mnemonic names

Defined registers are:

| Symbol | Value                    |
|--------|--------------------------|
| ------ | Registers                |
| R0     | 0,4,C'g'                 |
| R1     | 1,4,C'g'                 |
| R2     | 2,4,C'g'                 |
| R3     | 3,4,C'g'                 |
| R4     | 4,4,C'g'                 |
| R5     | 5,4,C'g'                 |
| R6     | 6,4,C'g'                 |
| R7     | 7,4,C'g'                 |
| R8     | 8,4,C'g'                 |
| R9     | 9,4,C'g'                 |
| R10    | 10,4,C'g'                |
| R11    | 11,4,C'g'                |
| R12    | 12,4,C'g'                |
| R13    | 13,4,C'g'                |
| R14    | 14,4,C'g'                |
| R15    | 15,4,C'g'                |
| ------ | Access registers         |
| AR0    | 0,4,C'a'                 |
| AR1    | 1,4,C'a'                 |
| AR2    | 2,4,C'a'                 |
| AR3    | 3,4,C'a'                 |
| AR4    | 4,4,C'a'                 |
| AR5    | 5,4,C'a'                 |
| AR6    | 6,4,C'a'                 |
| AR7    | 7,4,C'a'                 |
| AR8    | 8,4,C'a'                 |
| AR9    | 9,4,C'a'                 |
| AR10   | 10,4,C'a'                |
| AR11   | 11,4,C'a'                |
| AR12   | 12,4,C'a'                |
| AR13   | 13,4,C'a'                |
| AR14   | 14,4,C'a'                |
| AR15   | 15,4,C'a'                |
| ------ | Floating point registers |
| FPR0   | 0,8,C'f'                 |
| FPR2   | 2,8,C'f'                 |
| FPR4   | 4,8,C'f'                 |
| FPR6   | 6,8,C'f'                 |
| ------ | Control registers        |
| CR0    | 0,4,C'c'                 |
| CR1    | 1,4,C'c'                 |
| CR2    | 2,4,C'c'                 |
| CR3    | 3,4,C'c'                 |
| CR4    | 4,4,C'c'                 |
| CR5    | 5,4,C'c'                 |
| CR6    | 6,4,C'c'                 |
| CR7    | 7,4,C'c'                 |
| CR8    | 8,4,C'c'                 |
| CR9    | 9,4,C'c'                 |
| CR10   | 10,4,C'c'                |
| CR11   | 11,4,C'c'                |
| CR12   | 12,4,C'c'                |
| CR13   | 13,4,C'c'                |
| CR14   | 14,4,C'c'                |
| CR15   | 15,4,C'c'                |

## Syntax

``` hlasm
         MAPREGS ,
```
