# LA macro

This macro replaces the LA instruction

Filling a register with a zero value is replaced by XR

## Syntax

``` hlasm
&LABEL   LA    ,                       * Parameters in SYSLIST
```

The syntax for the LA macro is identical to that of the LA instruction.

## Macro code

The [LA macro](../bxamac/LA.mac) is in the BXAmac folder.
