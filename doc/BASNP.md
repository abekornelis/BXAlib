# BASNP macro

Branch and Save on Not-Plus condition

## Syntax

``` hlasm
&LABEL   BASNP &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *                                
```
