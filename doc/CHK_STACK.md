# CHK_STACK macro

Check that the structured programming stack is empty 

## Syntax

``` hlasm
         CHK_STACK &RESET=NO           * NO or YES    
```

## RESET

Inidicates whether or not to reset the structured programming stack.

## Macro code

The [CHK_STACK macro](../bxamac/PGM0.mac) is in the BXAmac folder.
