# CHKMAP macro

Checking function for parameters of MAP-macro's

## Syntax

``` hlasm
&LABEL   CHKMAP &DSECT=,               * YES or NO                     *
               &CB=,                   * Control block to generate     *
               &PRFX=                  * Prefix to use
```

## Macro code

The [CHKMAP macro](../bxamac/CHKMAP.mac) is in the BXAmac folder.
