# BALNP macro

Branch and Link on Not-Plus condition

## Syntax

``` hlasm
&LABEL   BALNP &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *                                
```
