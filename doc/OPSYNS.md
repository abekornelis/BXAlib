# OPSYNS macro

This macro generates OPSYNs for a list of opcodes

## Syntax

``` hlaasm
         OPSYNS ,                      * Arguments in &SYSLIST
```

## SYSLIST

each entry specifies an opcode or directive which will
be replaced by its underscore-prefixed equivalent

The [OPSYNS macro](../bxamac/OPSYNS.mac) is in the BXAmac folder.
