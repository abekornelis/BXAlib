# IF$ALC macro

Allocate an available general purpose register

This is a sub-macro for use by the [IF$ macro](IF$.md) only

## Syntax

``` hlasm
&LABEL   IF$ALC &REG                   * Register to protect
```

## Macro code

The [IF$ALC macro](../bxamac/IF$ALC.mac) is in the BXAmac folder.
