# LABEL macro

This macro generates a label on a specified boundary

## Syntax

``` hlasm
&LABEL   LABEL &ALIGN                  * Alignment code
```

## ALIGN

If omittted, no alignment is enforced.

Values of H, F, or D are accepted to enforce alignment
on a Halfword, Fullword, or Doubleword boundary, respectively.
