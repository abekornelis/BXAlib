# RLTORG macro

Create a remote literal pool. This macro creates a literal pool for data
elements defined using the [RDATA macro](RDATA.md).
Just like the [LTORG macro](LTORG.md) generates a literal pool for ordinary literals.

## Syntax

``` hlasm
&LABEL   RLTORG ,
```

## Macro code

The [RLTORG macro](../bxamac/RLTORG.mac) is in the BXAmac folder.
