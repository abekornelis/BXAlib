# USEDREGS macro

This macro lists all registers currently in use

## Syntax

``` hlasm
         USEDREGS &OPT                 * DETAILS or nothing
```

## OPT

If specified as `DETAILS`, active entries from the
`BXA_USELBL` and `BXA_USEREG` tables will be shown

## Macro code

The [USEDREGS macro](../bxamac/USEDREGS.mac) is in the BXAmac folder.
