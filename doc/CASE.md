# CASE macro

- Structured programming macro: CASE
- Combines with ELSE and ENDCASE to conditionally execute code-blocks

## Syntax

``` hlasm
&LABEL   CASE  &NEST=NO                * YES to force nesting
.*                                     * Condition in &SYSLIST
```
