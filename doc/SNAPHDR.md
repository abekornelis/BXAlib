# SNAPHDR macro

This macro generates a header-text for use with STRHDR operand of the SNAP macro

## Syntax

``` hlasm
&LABEL   SNAPHDR &TXT                  * Text for the header
```
## LABEL

Specifies the label for the text-parameter

## TXT

Specifies the text to be used, enclosed in single quotation marks

## Macro code

The [SNAPHDR macro](../bxamac/SNAPHDR.mac) is in the BXAmac folder.
