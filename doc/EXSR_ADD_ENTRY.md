# EXSR_ADD_ENTRY macro

The EXSR_ADD_ENTRY is used by the [EXSR macro](EXSR.md) to manage the call stack.

## Syntax

``` hlasm
         EXSR_ADD_ENTRY &NAME,&TYPE    * Name and type of subroutine
```

## NAME

Name of the subroutine

## TYPE

Type of subroutine: `INT`ernal or `EXT`ernal.

## Macro code

The [EXSR_ADD_ENTRY macro](../bxamac/EXSR0.mac) is in the BXAmac folder.
