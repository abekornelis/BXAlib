# SETMODE0 macro

This macro creates the [SAVE_PSWKEY macro](SAVE_PSWKEY.md)

## Syntax

``` hlasm
         SETMODE0 ,
```

The SETMODE0 macro takes no arguments.
It is intended for use by the [SETMODE macro](SETMODE.md)

## Macro code

The [SETMODE0 macro](../bxamac/SETMODE0.mac) is in the BXAmac folder.
