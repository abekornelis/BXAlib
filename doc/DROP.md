# DROP macro

This macro replaces the assembler's DROP statement

## Syntax

``` hlasm
         DROP  ,                       * Reg(s) and/or label(s)
```

## SYSLIST

specifies the labels and/or registers to drop.
If empty all registers will be dropped.

The [DROP macro](../bxamac/DROP.mac) is in the BXAmac folder.
