# MAPDSCB macro

This macro maps the Data Set Control Block formats 1 thru 5

The DSCB-formats are documented in DFSMSdfp Advanced Services Topic 1.1.1 ff

## Syntax

``` hlasm
&LABEL   MAPDSCB ,                     * No parms
```

## Macro code

The [MAPDSCB macro](../bxamac/MAPDSCB.mac) is in the BXAmac folder.
