# BALM macro

- Branch and Link on Mixed condition
- Branch and Link on Minus condition

## Syntax

``` hlasm
&LABEL   BALM  &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *
```
