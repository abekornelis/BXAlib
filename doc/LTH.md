# LTH macro

Load and Test signed halfword into register

## Syntax

``` hlasm
&LABEL   LTH   &REG,                   * Register to load              *
               &LOC                    * Location of value
```


## REG

specifies the register to load

## LOC

specifies the location that holds the value to be loaded
