# BALNE macro

Branch and Link on Not-Equal condition

## Syntax

``` hlasm
&LABEL   BALNE &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *
```
