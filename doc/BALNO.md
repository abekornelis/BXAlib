# BALNO macro

- Branch and Link on Not-Ones condition
- Branch and Link on No-Overflow condition

## Syntax

``` hlasm
&LABEL   BALNO &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *                                
```
