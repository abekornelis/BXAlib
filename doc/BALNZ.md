# BALNZ macro

Branch and Link on Non-Zero condition

## Syntax

``` hlasm
&LABEL   BALNZ &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *
```
