# CMDTXT macro

This macro generates a command-text for use with MGCRE macro

## Syntax

``` hlasm
&LABEL   CMDTXT &TXT                   * Text for the command
```

## LABEL

Specifies the label for the text-parameter

## TXT

Specifies the text to be used, enclosed in single quotation marks

## Macro code

The [CMDTXT macro](../bxamac/CMDTXT.mac) is in the BXAmac folder.
