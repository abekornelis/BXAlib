# BALNM macro

- Branch and Link on Not-Mixed condition
- Branch and Link on Not-Minus condition

## Syntax

``` hlasmn
&LABEL   BALNM &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *                                
```
