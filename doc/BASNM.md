# BASNM macro

- Branch and Save on Not-Mixed condition
- Branch and Save on Not-Minus condition

## Syntax

``` hlasm
&LABEL   BASNM &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *
```
