# EXTRN macro

This macro replaces the assembler's EXTRN statement.
It implements the overrides, as specified by preceding invocations
of the [EXTRNOVR macro](EXTRNOVR.md).

## Syntax

``` hlasm
&LABEL   EXTRN ,                       * Arguments in &SYSLIST
```

The syntax for the EXTRN macro is identical to that of the EXTRN instruction.

## Macro code

The [EXTRN macro](../bxamac/EXTRN.mac) is in the BXAmac folder.
