# BALO macro


- Branch and Link on Ones condition
- Branch and Link on Overflow condition

## Syntax

``` hlasm
&LABEL   BALO  &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *                                
```
