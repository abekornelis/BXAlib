# BASNZ macro

Branch and Save on Non-Zero condition

## Syntax

``` hlasm
&LABEL   BASNZ &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *
```
