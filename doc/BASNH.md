# BASNH macro

Branch and Save on Not-High condition

## Syntax

``` hlasm
&LABEL   BASNH &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *                                
```
