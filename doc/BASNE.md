# BASNE macro

Branch and Save on Not-Equal condition

## Syntax

``` hlasm
&LABEL   BASNE &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *                                
```
