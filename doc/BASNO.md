# BASNO macro

- Branch and Save on Not-Ones condition
- Branch and Save on No-Overflow condition

## Syntax

``` hlasm
&LABEL   BASNO &REG,                   * Register for return address   *
               &LOC,                   * Branch target or (reg)        *
               &TYPE=LOCAL             *
```
